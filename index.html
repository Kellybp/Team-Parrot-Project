Alt Text Generator


How To:

  1. Choose an image to upload (we do not keep any images)
  2. Upload the image via either drag and drop inside the blue dotted section or by selecting "Choose File"
    Accepted images: All image types are accepted.
  3. Select the "Run Generation" button to generate alt text
  4. The generated text comes from Google (nouns), IBM (full alt text), and a machine learning alorithm (full alt text)
  5. Use the section called "Build Your Alt Text" to develop your alt text and copy it using the " Copy Text" button
